## Rodando localmente

Clone the project

```bash
  git clone https://github.com/Z4Dev/RockeSeat-DevLinks.git
```

Enter the project directory

```bash
  cd folder_name
```

Install the dependencies

```bash
  yarn install or npm install
```

Start in development mode

```bash
  yarn dev or npm run dev
```
